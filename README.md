# beatpoker_publish

对本软件有任何意见可以通过Issues 提交，本人会及时处理。  
提交的Issues主要分2种：  
  
1 提交机器人策略  
提交机器人策略的意见，建议按照如下格式提交：  
回合     :xx  
公共牌   :xx,xx,xx  
牌桌位置 :xx,xx  
Button  :位置  
玩家手牌 :xx  
玩家动作 :xx  
机器人手牌 :xx  
期望机器人动作:xx  
其他补充 :xx  
  
如：  
回合     :flop  
公共牌   :红心A,红心K,红心Q  
牌桌位置 :1号位--玩家，2号位--机器人  
Button  :2号位  
玩家手牌 :梅花A,方块Q  
玩家动作 :3 Bet  
机器人手牌 :红心J,黑桃9  
期望机器人动作:Allin  
其他补充 :玩家前置位raise，机器人reraise之后玩家还能 3Bet，机器人红花听牌可以堵全家  
  
2 其他意见
